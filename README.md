# Interactive Tic-Tac-Toe in C with SDL3

## Description

This project is an interactive **Tic-Tac-Toe game** developed in **C** with a graphical interface using **SDL3**.  
The game offers multiple modes: 
- Player vs Player  
- Player vs AI (3 difficulty levels)

This project was developed as part of programming coursework to practice:  
- Game logic in C  
- User input handling  
- Simple graphical interface implementation  

---

## Features

- Graphical interface with SDL3  
- Automatic win condition detection  
- User input management  
- Game modes: Player vs Player and Player vs AI  
- Three AI difficulty levels  

---

## Technologies Used

- **Language**: C  
- **Graphics library**: SDL3
